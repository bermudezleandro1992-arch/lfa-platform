# Carpeta de entrenamiento — eFootball

Agrega aquí screenshots reales de pantallas de resultado de eFootball para que el BOT IA las use como referencia.

## Qué poner

- Fotos de la pantalla final del partido (Resultado / Match Result)
- Que se vea claramente el marcador
- Que se vean los nombres/IDs de ambos jugadores (eFootball ID)
- Variedad: distintos marcadores, distintas plataformas (PS5, PC, Mobile)

## Nomenclatura sugerida

```
resultado_2-1.jpg
resultado_0-0_pk.jpg
resultado_5-2.jpg
```

## Zonas que el BOT analiza

1. **Marcador** — números grandes en el centro
2. **Nombres de jugador** — eFootball Player ID de cada competidor
3. **Logo del juego / UI** — para detectar que es eFootball y no otra pantalla
4. **Estado** — "Full Time" / "PK" para confirmar resultado final real

> No uses screenshots editados. El BOT aprende de imágenes reales.
