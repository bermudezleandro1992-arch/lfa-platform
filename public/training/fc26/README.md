# Carpeta de entrenamiento — FC 26

Agrega aquí screenshots reales de pantallas de resultado de FC 26 para que el BOT IA las use como referencia.

## Qué poner

- Fotos de la pantalla final del partido (Full Time / Resultado Final)
- Que se vea claramente el marcador
- Que se vean los nombres/IDs de ambos jugadores
- Variedad: local vs visitante, distintos marcadores, distintas condiciones de luz/calidad

## Nomenclatura sugerida

```
resultado_3-0.jpg
resultado_1-1_overtime.jpg
resultado_penalties.jpg
```

## Zonas que el BOT analiza

1. **Marcador** — números grandes en el centro
2. **Nombres de jugador** — arriba de cada equipo (EA ID / gamertag)
3. **Logo del juego / UI** — para detectar que es FC 26 y no otra pantalla
4. **Tiempo** — 90' o penales para confirmar que es resultado final

> No uses screenshots editados. El BOT aprende de imágenes reales.
